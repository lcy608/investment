# Kiwoom OpenAPI+ Project

This is a basic starter project for using Kiwoom Securities OpenAPI+ with Python 32-bit.

## Prerequisites
1. Installed Kiwoom OpenAPI+ (KOA Studio).
2. Connected/Applied to Kiwoom OpenAPI+ service.
3. **Python 32-bit environment** (Required because the Kiwoom OCX component is 32-bit only). 

An automated installation script has already downloaded Python 3.11 (32-bit) and set up a virtual environment here:
`d:\python output\kiwoom_project\venv`

## Running The Project

프로젝트를 실행하는 가장 쉬운 방법:
`d:\python output\kiwoom_project` 폴더로 이동한 뒤, **`run.bat`** 파일을 더블클릭해서 실행하세요!

(또는 터미널에서 `.\run.bat` 을 입력하시면 됩니다)

A UI window will pop up. Click **"Login to Kiwoom"**. If everything is set up correctly, Kiwoom's login window will appear. Enter your credentials, and you'll see successful connection logs with your account number!
